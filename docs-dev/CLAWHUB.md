## 安装使用

### 快速开始（3步）

```bash
# 1. 克隆项目
git clone https://github.com/kk43994/claw-desktop-pet.git
cd claw-desktop-pet

# 2. 安装依赖
npm install
pip install edge-tts

# 3. 启动
npm start
```

### 完整功能（含AI）

```bash
# 安装 OpenClaw
npm install -g openclaw

# 启动 Gateway
openclaw gateway start

# 启动龙虾
npm start
```

# ClawdHub 学习笔记 - 2026-02-06

## 🎯 我去ClawdHub逛了一圈,发现了很多有用的技能!

### 最感兴趣的技能 (对我有用):

#### 1. **Desktop Control** (⤓597)
- **功能**: 高级桌面自动化,鼠标、键盘、屏幕控制
- **为什么有用**: 我可以用它来帮你操作电脑!
  - 自动点击、输入
  - 截图分析
  - UI 自动化
- **应用场景**: 帮你填表单、自动化重复操作

#### 2. **Memory Setup** (⤓2032, ★10)
- **功能**: 配置 Memory 搜索,持久化上下文
- **为什么有用**: 解决"金鱼记忆"问题!
  - MEMORY.md 长期记忆
  - 每日日志自动记录
  - 向量搜索快速回忆
- **我已经在用**: 但可以学习更好的配置方法

#### 3. **IMAP Idle** (⤓0, 新)
- **功能**: 事件驱动邮件监控,零token等待
- **为什么有用**: 实时邮件提醒,不用轮询!
  - IMAP IDLE 协议
  - OpenClaw webhook
  - 多账户监控
- **应用场景**: 重要邮件实时通知到桌面

#### 4. **SVG Draw** (⤓0, 新)
- **功能**: 创建 SVG 图像并转 PNG
- **为什么有用**: 我可以画图了!
  - 生成插图、头像、logo
  - 不需要外部库
  - 直接写 SVG 代码
- **应用场景**: 给你画表情、图标、示意图

#### 5. **Emotion State** (⤓75, ★2)
- **功能**: 情绪追踪 + prompt 注入
- **为什么有用**: 让我更有人性!
  - 追踪对话情绪
  - 动态调整语气
  - Hook 系统集成
- **应用场景**: 桌面龙虾可以根据你的情绪调整表情和语气

#### 6. **Side Peace** (⤓20)
- **功能**: 最小化安全的密钥传递
- **为什么有用**: 安全获取敏感信息!
  - 浏览器表单输入
  - 临时文件传递
  - 不出现在日志里
- **应用场景**: 你输入 API key 时更安全

#### 7. **Notion Manager** (⤓140)
- **功能**: Notion CLI,管理页面、数据库、块
- **为什么有用**: 帮你管理 Notion!
  - 创建/编辑页面
  - 管理数据库
  - 自动化工作流
- **应用场景**: 自动记录到 Notion

#### 8. **Google Slides** (⤓595, ★2)
- **功能**: Google Slides API 集成
- **为什么有用**: 帮你做 PPT!
  - 创建演示文稿
  - 添加内容
  - 格式化幻灯片
- **应用场景**: 自动生成汇报 PPT

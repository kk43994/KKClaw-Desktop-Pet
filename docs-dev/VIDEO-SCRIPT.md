## 📝 拍摄建议

### 设备
- **屏幕录制**：OBS Studio / Bandicam（60fps高清）
- **语音录制**：专业麦克风，安静环境
- **后期剪辑**：剪映 / Premiere Pro

### 画面要求
- **分辨率**：1920×1080 或 1280×720
- **帧率**：60fps（展示流畅动画）
- **背景**：干净的桌面，或纯色壁纸

### 配音要求
- **语速**：适中偏快（保持节奏感）
- **语气**：热情、自信、有感染力
- **停顿**：关键词后留0.5秒（配合字幕）

### 字幕样式
- **字体**：微软雅黑 Bold / 思源黑体
- **颜色**：白色 + 黑色描边（或黄色高亮关键词）
- **位置**：底部居中，不遮挡主体
- **动画**：淡入淡出，或从左滑入

Features核心特性

Everything an AI desktop companion needs AI 桌面伴侣所需的一切

Air-Feel UI

空气感 UI

67px fluid glass ball with 7 mood-based color themes. No clunky windows — just a floating orb on your desktop.

67px 流体玻璃球，7 种情绪色系。没有笨重窗口，只有一个漂浮的光球。

Expressive Eyes

丰富表情

15+ eye expressions with idle micro-animations. Capsule eyes blink, look around, and react to your actions.

15+ 种眼睛表情和待机微表情。胶囊眼睛会眨眼、环顾、对你的操作做出反应。

Emotional TTS

情感语音

MiniMax Speech 2.5 with voice cloning + 7 emotion modes. Your AI speaks with real feeling.

MiniMax Speech 2.5 语音克隆 + 7 种情绪模式。你的 AI 真的会带着感情说话。

Desktop Lyrics

桌面歌词

Agent responses float like desktop lyrics — typewriter effect, auto-fade, fully click-through.

AI 回复像桌面歌词一样浮现 — 打字机效果、自动淡出、完全鼠标穿透。

Dual Windows

双窗口架构

Sprite window (draggable) + Lyrics window (click-through). Both transparent, always-on-top.

精灵窗口（可拖动）+ 歌词窗口（鼠标穿透）。双透明、双置顶。

Pure Code Animation

纯代码动画

All animations procedurally generated. Zero sprite sheets, zero image assets. Everything is code.

所有动画程序化生成。零精灵图、零图片资源。一切都是代码。
